Dermatologist AI Source code

Skin cancer is the most common cancer in the world, just in the U.S.A, there are 5.4 million cases every year. Melanoma is the kind that kills people. It comes in multiple stages, from 0 to 4. The survival probability for stage four is 15-20%, and for stage zero is 99-100%, so early detection is paramount to not dying from cancer.

This project helps to diagnose melanoma. It distinguishes the malignant skin tumor from two types of benign lesions: nevus and seborrheic keratosis.



Semantic Segmentation Source code

Imagine that you ordered tacos, two of pastor and six of bistec, all on the same plate; The waiter comes out, so you wonder if what he is carrying are tacos; He gives to you, and you quickly figure out where the bistec's are located in the plate. In computer vision, classification solves the what, and semantic segmentation the where 'exactly'.

This project classifies pixel by pixel 32 different types of objects with a state-of-art performance on the CamVid dataset.



YOLO v3 Implementation Source code

In computer vision, localization means to know where an object is located in an image, that is, predict a class from an object and a bounding box for that object. You Only Look Once (YOLO) is a real-time object detection algorithm that instead of locating objects perfectly (like the semantic segmentation project) it prioritizes speed and recognition. This is important for any application that relies on looking at the proximity of two or more objects.



Mariya Web site

There is one of the oldest indigenous community of the history in México, erroneously known as Huicholes. They live in the highest part of the Sierra Madre Occidental where the access to new technologies is almost impossible; this forces new generations to move to the big cities and face a language problem that threatens the extinction of their mother tongue. So, I collaborated to create Mariya, a project that seeks to put an end to their communication barrier through a translator device.
